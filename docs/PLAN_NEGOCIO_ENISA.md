## 1. RESUMEN EJECUTIVO

### 1.1 Concepto del Negocio

**Aura World** es una aplicación móvil de gaming que combina:
- **Realidad Aumentada (AR)**: Captura de objetos virtuales en el mundo real
- **Move-to-Earn (M2E)**: Recompensas económicas por actividad física
- **Blockchain/NFT**: Economía descentralizada en Polygon

**Propuesta de valor única**: "Camina, colecciona, gana dinero real"

## 2. DESCRIPCIÓN DEL PRODUCTO

### 2.1 Funcionalidades Principales

#### AR Crystal Hunting
- Usuarios caminan por el mundo real
- Cristales virtuales aparecen en ubicaciones GPS
- Cámara AR para captura inmersiva
- Sistema anti-cheat con validación backend

#### Economía NFT
- Cristales = NFTs en Polygon
- Raridades: Common, Rare, Epic, Legendary
- Marketplace P2P para trading
- Subastas en vivo con anti-sniping

#### PvP Dueling
- Batallas en tiempo real
- Apuestas de NFTs
- Sistema ELO de ranking

#### Tokenomics (AURA Token)
- Utilidad: Staking, governance, compras in-game
- Modelo deflacionario (quema en transacciones)
- Fair launch (sin pre-venta privada masiva)

### 2.2 Stack Tecnológico

| Capa | Tecnología |
|------|------------|
| Frontend | React + TypeScript + Framer Motion |
| Backend | NestJS + MongoDB |
| Blockchain | Solidity 0.8.20 + Hardhat + Polygon |
| AR | HTML5 Camera API + Geolocation |
| Monitoring | Sentry SDK v8 |

### 2.3 Diferenciación Competitiva

| Competidor | Debilidad | Ventaja Aura World |
|------------|-----------|-------------------|
| STEPN | Solo caminar, sin AR | AR inmersivo + gameplay |
| Pokemon GO | Sin economía real | Ganancias reales en crypto |
| Atlas Earth | Pasivo, sin blockchain | Activo + NFT ownership |
