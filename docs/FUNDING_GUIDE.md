## 📋 Resumen Ejecutivo

## 🇪🇸 Ayudas Gubernamentales España

### 1. ENISA - Préstamos Participativos (⭐ RECOMENDADO)

| Característica | Detalle |
|----------------|---------|
| **Organismo** | ENISA (Ministerio de Industria) |
| **Importe** | €25,000 - €1,500,000 |
| **Tipo** | Préstamo participativo (sin garantías personales) |
| **Plazo** | Hasta 9 años |
| **Interés** | Variable según rentabilidad empresa |
| **Requisitos** | Empresa innovadora, plan viable |
| **Estado 2025** | €200M disponibles (abierto todo el año) |
| **Tiempo respuesta** | ~35 días |

**URL**: https://www.enisa.es

**Lines de financiación aplicables**:
- Línea "Startups y PYMES" (nueva 2025)
- Línea específica sector audiovisual (incluye videojuegos)

### 2. Ministerio de Cultura - Subvenciones Videojuegos

| Característica | Detalle |
|----------------|---------|
| **Importe** | Hasta €200,000 |
| **Tipo** | Subvención a fondo perdido |
| **Convocatoria** | Anual (normalmente Q1) |
| **Presupuesto total** | €8M (Plan de Recuperación) |
| **Requisitos** | Desarrollo de videojuegos/metaverso |

**Para 2025**: €5M del fondo Next Generation EU.

### 3. GameBCN - Incubadora Barcelona

| Característica | Detalle |
|----------------|---------|
| **Importe** | €3,000 + formación gratuita |
| **Tipo** | Programa de incubación (6 meses) |
| **Incluye** | 60h formación + 36h mentoría |
| **Próxima edición** | Febrero 2025 |
| **Deadline pasado** | 31 Dic 2024 (siguiente: Nov 2025) |

**URL**: https://gamebcn.co

### 4. ICEX - Internacionalización

| Característica | Detalle |
|----------------|---------|
| **Programa** | "Games from Spain" |
| **Tipo** | Apoyo a feria (GDC, Gamescom) |
| **Beneficios** | Stand colectivo, networking |
| **Spain Audiovisual Hub** | €20M fondos Next Gen EU |

**URL**: https://www.icex.es

### 5. Cataluña - Incentivos Regionales

- Subvenciones I+D para videojuegos
- Incubadoras de creatividad digital
- Incentivos reembolsables para desarrollo

**URL**: https://videojocscatalans.cat
